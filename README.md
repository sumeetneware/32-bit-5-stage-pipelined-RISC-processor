# 2-bit-5-stage-pipelined-RISC-processor
This project presents the design and Verilog implementation of a 32-bit 5-stage pipelined RISC processor with IF, ID, EX, MEM, and WB stages. The processor supports a basic instruction set (`ADD`, `SUB`, `AND`, `OR`, `LW`, `SW`, `BEQ`) and includes hazard detection, forwarding, branch flush logic, and direct-mapped instruction/data caches.
